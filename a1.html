// Initializing the integer variable timer which represents the delay time value.
int timer = 250;

void setup() {
  // Using a for loop to initialize pins 11, 12 and 13 as output.
  for (int thisPin = 11; thisPin < 14; thisPin++) {
    pinMode(thisPin, OUTPUT);
  }
}

void loop() {
  // Using a for loop to loop from pin 11 to pin 13.
  for (int thisPin = 11; thisPin < 14; thisPin++) {
    // Turning the pin on and its associated LED.
    digitalWrite(thisPin, HIGH);
    delay(timer);
    // Turning the pin off and its associated LED.
    digitalWrite(thisPin, LOW);
  }

  // Using a for loop to loop from pin 13 to pin 11 (backwards).
  for (int thisPin = 13; thisPin >= 11; thisPin--) {
    // Turning the pin on and its associated LED.
    digitalWrite(thisPin, HIGH);
    delay(timer);
    // Turning the pin off and its associated LED.
    digitalWrite(thisPin, LOW);
  }
}

Answers to Additional Questions

1. Draw a graph where the X axis is time and the Y axis is voltage across the LEDs. Draw a line for each of your 3 LEDs.

2. How many LEDs could you blink independently with your Arduino? How much current would that draw?

The Arduino has about 13 usable pins, so in theory it could blink 13 LEDs independently. To calculate the current draw, we can use Ohm's Law (I = V/R). Assuming each LED has a voltage of 2V and we are using a 220-ohm resistor for each LED, the current through each LED would be approximately 9.09 mA (I = (5V - 2V) / 220 ohms). Therefore, for 13 LEDs, the total current draw would be approximately 118.17 mA.

3. How fast do you need to blink your LEDs until you no longer can tell that they are blinking?

Setting the timer variable value to about 8 ms in the code would achieves the effect where you no longer tell that the LED's are blinking.
